Minimal Task Manager app for students. Features: register/login, CRUD on tasks, logout.


## Setup
1. Create a Back4App app and copy Application ID and Client Key.
2. Replace credentials in `lib/config/parse_config.dart`.
3. Run:
```bash
flutter pub get
flutter run
```


## Files
- `lib/` contains main app code.
- `pubspec.yaml` lists dependencies.
